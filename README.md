## 🌟 Features 🌟

-   🚓 List of all police forces in the UK
-   🔍 Search functionality to filter the list of police forces
-   📝 Detailed view for each police force, including:
    -   📛 Name and description of the force
    -   👮‍♂️ Senior officers
        -   📖 Name
        -   🎖️ Rank
            -   🔎 Filter via rank
        -   📚 Bio (if set)
    -   📞 Contact information
        -   📱 Social media
        -   🌐 Website
-   📱 Responsive design that works on both desktop and mobile browsers
-   📊 Uses [UK Police API](https://data.police.uk/docs/) for the data
